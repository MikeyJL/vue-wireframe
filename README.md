# vue-wireframe

A vue component that helps you plan UI layouts and lets your change the CSS easily.

## Usage

```
import Vue from 'vue'
import Wireframe from 'vue-wireframe'

Vue.use(Wireframe)
```

Using the component:

```
<div>
  <wireframe />
</div>
```

Or pass in child html elements

```
<wireframe>
  <h1>Hello world!</h1>
  <p>This will appear inside the wireframe</p>
</wireframe>
```
